Customer Churn Prediction

Machine Learning Project

Predicting customer churn using advanced Machine Learning algorithms and data preprocessing techniques to help telecommunication companies improve customer retention.

Project Overview

Customer Churn Prediction is a Machine Learning project developed to identify customers who are likely to stop using a telecom company's services. The project uses customer demographics, account details, service subscriptions and billing information to predict future churn behavior. By identifying at-risk customers early, businesses can implement retention strategies and reduce revenue loss.

Problem Statement

Customer acquisition is significantly more expensive than customer retention. Telecom companies lose millions in revenue due to customer churn each year. The goal of this project was to build a predictive model that accurately identifies customers likely to leave the service before churn actually occurs.

Handling Class Imbalance

Problem

The dataset was highly imbalanced because the number of customers who stayed was much larger than those who churned.


Solution

Applied SMOTEENN (Synthetic Minority Oversampling Technique + Edited Nearest Neighbors).

  • Balanced class distribution
  • Generated synthetic minority samples
  • Removed noisy observations
  • Improved model performance
