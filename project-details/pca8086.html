PCA Using 8086 Assembly Language

Research-Oriented Computer Architecture Project

Implementation of Approximate Principal Component Analysis using Oja's Learning Rule on Intel 8086 Architecture for dimensionality reduction and machine learning applications.

Project Overview

Principal Component Analysis (PCA) is one of the most important dimensionality reduction techniques used in Machine Learning and Data Science. This project explores the implementation of an Approximate PCA algorithm on Intel 8086 Assembly Language, demonstrating how machine learning concepts can be executed on low-level computer architectures. The project combines concepts from Machine Learning, Linear Algebra, Computer Organization & Architecture, Assembly Programming and Optimization.

Problem Statement

Modern PCA implementations typically rely on high-level programming languages and powerful computing resources. The objective of this project was to investigate whether dimensionality reduction techniques could be implemented on resource-constrained hardware such as Intel 8086 microprocessors. The project demonstrates how machine learning concepts can be adapted for low-level computing environments.

Project Objectives

What is PCA?

Principal Component Analysis (PCA) is a statistical technique used to reduce the number of features in a dataset while preserving maximum information. PCA transforms the original data into a new coordinate system called Principal Components. These components capture the most important patterns and variations in the dataset.


Benefits of PCA

  • Dimensionality Reduction
  • Noise Reduction
  • Feature Extraction
  • Data Compression
  • Improved Visualization
  • Faster Machine Learning Models

Oja's Learning Rule

Oja's Learning Rule is an unsupervised learning algorithm used to estimate the first principal component of a dataset. Instead of performing complex matrix decompositions, Oja's Rule iteratively updates weight vectors to approximate the dominant eigenvector. This makes it suitable for implementation on resource-constrained hardware such as the Intel 8086.


Advantages

  • Memory Efficient
  • Computationally Lightweight
  • Suitable for Embedded Systems
  • Incremental Learning Capability
